HostsXEditor
============
2010 年代的 Hosts 文件编辑工具。

需求
----
* AutoHotkey_H Unicode 32-bit 1.1.09.03
* Ahk2Exe for AutoHotkey 1.1.09.03
* Microsoft Windows XP/Vista/7/8 or Server 2003/2005/2008/2012
